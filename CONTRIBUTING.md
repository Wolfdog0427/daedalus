# Contributing to Daedalus — Kernel Era

The Kernel era is sealed governance architecture.

---

## Golden rule

If a change alters governance, sovereignty, epistemic behavior, continuity, or node trust,  
it is **out of scope**.

---

## Allowed

- Apps  
- Panels  
- Tools  
- Styling  
- Tests  
- Docs  

---

## Forbidden

- KernelShell  
- Kernel  
- Invariants  
- Halo  
- Crown  
- Throne  
- Epistemic filters  
- Sovereignty logic  
- Node trust  
- Continuity semantics  
- Integration Pass semantics
